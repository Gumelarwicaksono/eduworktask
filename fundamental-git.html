perintah dasar git

  • git init ("menginisialisasi sebuah folder untuk di jadikan sebuah repo")
  • git config ("melakukan configurasi awal saat pertamakali menginstal git ")
  • git status ("digunakan untuk mengetahui status dari sebuah repositori lokal")
  • git add . ("untuk menambakan file baru pada repo yang dipilih")
  • git commit -m "isi commit" ("untuk mengirimkan perubahan file setelah di commit ke repositori")
  • git clone ("untuk membuat salinan repositori dari github / gitlub ke lokal")
  • git remote ("untuk melihat remote git yang ada jika ingin membuat remote tambahkan {add nama_remote link-dari-github/gitlab} ")
  • git push ("untuk mengirimkan perubahan file setelah di commit ke remote repositori")
  • git pull ("untuk memindahkan repositori ke lokal computer")
